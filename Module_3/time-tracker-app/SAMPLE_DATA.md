## Testing Scenarios

### Scenario 1: Validation Test
```
1. Add activities totaling 1200 minutes
2. Try to add 300 more minutes (should fail)
3. Should show: "Only 240 minutes remaining"
```

### Scenario 2: Multiple Categories
```
Add at least one activity from each category:
- Work, Study, Sleep, Exercise
- Entertainment, Food, Social, Other
```

### Scenario 3: Date Switching
```
1. Add activities for today
2. Switch to yesterday (should show "No data")
3. Switch back to today (should show activities)
```

### Scenario 4: Analytics Display
```
1. Add 8-10 activities
2. Click "Analyse"
3. Verify pie chart shows all categories
4. Verify bar chart shows all activities
5. Check summary cards are correct
```
